# 系统架构

> **版本**：v2.0 | **日期**：2026-06-07

---

## 1. 技术栈

| 组件 | 选型 | 用途 |
|------|------|------|
| 后端语言 | Python 3.13 | 爬虫、清洗、API、调度、统计 |
| Web 框架 | FastAPI | RESTful API，lifespan 生命周期 |
| ASGI | uvicorn | 运行 FastAPI |
| 数据库 | MySQL 8.0 | 结构化存储 |
| DB 驱动 | PyMySQL | Python ↔ MySQL |
| 统计加速 | numpy ≥ 1.26.0 | 向量化计算 |
| 定时调度 | APScheduler | 每 24h 爬取 |
| HTTP | requests | 下载外部 TXT |
| 容器化 | Docker + Compose | 3 容器编排 |
| 反向代理 | nginx:alpine | /api 代理到 app:8000 |
| 前端 | HTML5 + JS + ECharts 5 | 纯静态可视化 |
| 主题 | CSS 变量 + localStorage | 深色/浅色双主题 |

## 2. 部署架构

```
浏览器 ──→ nginx:80 ──→ / → 静态文件 (frontend/)
                     └── /api/* → proxy_pass → app:8000 (FastAPI)
                                                      └── MySQL:3306 (lottery_db)
```

- **Docker 生产**：3 容器（mysql + app + nginx），`docker compose up -d`
- **本地开发**：FastAPI StaticFiles 直出，`start.bat` 启动，无需 nginx
- **API 自适应**：`frontend/js/config.js` 根据 hostname 自动切换

## 4. 核心设计模式

### 4.1 RouterFactory（配置驱动路由）
```python
@dataclass
class LotteryConfig:
    key: str          # "dlt"/"ssq"/"7xc"/"pl5"/"pl3"
    table: str        # "lottery_dlt"/...
    tag: str          # 中文名称
    num_fields: list  # 号码字段列表
    ext_module: str   # 扩展统计模块路径
    uses_pos_stats: bool = False  # PL3/PL5 共用 pos_stats

RouterFactory.build(config) → APIRouter  # 自动生成 10 个端点
```

### 4.2 Crawler 工厂
`factory.py` 根据 `DATA_SOURCES` 动态生成 5 个 Crawler 子类，`__init__.py` 编排注册表。

### 4.3 前后区分栏（七星彩/大乐透/双色球）
基础统计中，有前后区概念的彩种自动拆分为两个独立图表。

## 5. 数据流

```
定时调度(每24h)
    └→ crawler.__init__.py
          └→ FactoryCrawler.fetch() → 下载TXT
                └→ Cleaner.parse() → 清洗为 records[]
                      └→ db.batch_insert() → MySQL
                            └→ API 查询/统计
                                  └→ ECharts 渲染
```
