# 数据设计

> **版本**：v2.0 | **日期**：2026-06-07

---

## 1. 数据库

- **库名**：`lottery_db`
- **引擎**：InnoDB
- **字符集**：`utf8mb4`，排序规则 `utf8mb4_unicode_ci`

## 2. 表结构（5 张表）

### 2.1 通用字段（5 表共有）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT PRIMARY KEY | 自增主键 |
| draw_num | VARCHAR(20) UNIQUE NOT NULL | 期号（如 "2025001"） |
| draw_date | DATE NOT NULL | 开奖日期 |
| sales_amount | VARCHAR(50) | 销售额（可空） |
| prize_pool | VARCHAR(50) | 奖池金额（可空） |
| created_at | TIMESTAMP DEFAULT CURRENT_TIMESTAMP | 入库时间 |

### 2.2 号码字段（按彩种）

#### lottery_dlt（超级大乐透）
```
front_1 TINYINT   -- 前区第1位 1-35
front_2 TINYINT   -- 前区第2位 1-35
front_3 TINYINT   -- 前区第3位 1-35
front_4 TINYINT   -- 前区第4位 1-35
front_5 TINYINT   -- 前区第5位 1-35
back_1  TINYINT   -- 后区第1位 1-12
back_2  TINYINT   -- 后区第2位 1-12
```

#### lottery_7xc（七星彩）
```
num_1   TINYINT   -- 前区第1位 0-9
num_2   TINYINT   -- 前区第2位 0-9
num_3   TINYINT   -- 前区第3位 0-9
num_4   TINYINT   -- 前区第4位 0-9
num_5   TINYINT   -- 前区第5位 0-9
num_6   TINYINT   -- 前区第6位 0-9
back_1  TINYINT   -- 后区 0-14
```

#### lottery_ssq（双色球）
```
red_1   TINYINT   -- 红球第1位 1-33
red_2   TINYINT   -- 红球第2位 1-33
red_3   TINYINT   -- 红球第3位 1-33
red_4   TINYINT   -- 红球第4位 1-33
red_5   TINYINT   -- 红球第5位 1-33
red_6   TINYINT   -- 红球第6位 1-33
blue_1  TINYINT   -- 蓝球 1-16
```

#### lottery_pl5（排列5）
```
num_1   TINYINT   -- 万位 0-9
num_2   TINYINT   -- 千位 0-9
num_3   TINYINT   -- 百位 0-9
num_4   TINYINT   -- 十位 0-9
num_5   TINYINT   -- 个位 0-9
```

#### lottery_pl3（排列3）
```
num_1   TINYINT   -- 百位 0-9
num_2   TINYINT   -- 十位 0-9
num_3   TINYINT   -- 个位 0-9
```

## 3. 索引

| 表 | 索引 |
|------|------|
| 全部 5 表 | UNIQUE INDEX `idx_draw_num` ON (draw_num) |
| 全部 5 表 | INDEX `idx_draw_date` ON (draw_date) |

## 4. 设计规则

- 号码拆分为独立 TINYINT 字段，**禁止逗号拼接**
- `draw_num` 作为业务主键，UNIQUE 约束 + INSERT IGNORE 去重
- 七星彩 `back_1` 字段名为历史遗留（非 num_7），范围 0-14 不同于前区 0-9
- PL3/PL5 无 sales_amount / prize_pool 数据，字段可空

## 5. numpy 交互

`numpy_utils.records_to_array(records, fields)` → shape (N, len(fields)) 的 float64 ndarray，缺失值填 NaN，跳过无效行。

## 6. 配置约定

`config.py` 中 `DATA_SOURCES[table] = {url, encoding}`，爬虫工厂据此动态生成 Crawler。环境变量覆盖数据库配置。
