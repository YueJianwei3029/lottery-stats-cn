# 前端方案设计

> **版本**：v2.0 | **日期**：2026-06-07

---

## 1. 技术栈

- **HTML5** + **CSS3**（CSS 变量双主题）+ **Vanilla JavaScript**
- **ECharts 5**（CDN 引入，`echarts.min.js` 本地副本也可用）
- 无框架依赖，纯静态页面

## 2. 页面结构

```
index.html
├── 顶栏：标题 "彩票数据统计与可视化系统" + 免责声明 + 主题切换按钮
├── 左侧导航：5 彩种选择（大乐透/七星彩/双色球/排列5/排列3）
├── 主区域
│   ├── Tab 1: 数据查询
│   │   ├── 日期选择 + 查询按钮 + 每页条数(10/20/50)
│   │   ├── 数据表格（开奖号码用彩色球渲染）
│   │   └── 分页控件
│   ├── Tab 2: 基础统计
│   │   ├── 号码频率条形图（前后区分栏或单图）
│   │   ├── 奇偶分布堆叠条形图
│   │   └── 大小分布堆叠条形图
│   ├── Tab 3: 扩展统计（子菜单：分布分析 / 走势分析）
│   │   ├── 热力图 + 冷热号
│   │   └── 和值跨度走势 + 奇偶比/大小比趋势
│   └── Tab 4: 高级统计（子菜单：散点分布 / 大数定律）
│       ├── 散点图（每位置独立图）
│       └── LLN 图（运行均值+置信带）
└── 移动端：底部固定导航栏
```

```js
var currentType = "lottery_dlt";       // 当前彩种
```

## 4. 关键函数清单

### 4.1 导航与视图

| 函数 | 说明 |
|------|------|
| `initNav()` | 左侧彩种列表点击切换 `currentType` |
| `initTabs()` | 4 Tab 切换 |
| `initTheme()` | 主题切换按钮，读写 localStorage |
| `switchTab(tab)` | 切换 Tab 后按需加载内容 |
| `initExtendSubMenu()` / `initAdvancedSubMenu()` | 子菜单按钮绑定 |

### 4.2 数据查询

| 函数 | 说明 |
|------|------|
| `queryData()` | 调用 `/query` API，渲染表格 |
| `renderTable(records)` | 渲染表格，号码用 `<span class="ball">` 包裹 |
| `buildNumLabels()` | 按彩种生成表头标签（前1、前2...） |
| `getHiddenFields()` | 七星彩隐藏 sales_amount/prize_pool |

### 4.3 基础统计

| 函数 | 说明 |
|------|------|
| `loadStats()` | 调用 `/stats` API |
| `buildNormalStats()` | 渲染普通彩种统计（含七星彩特化分支） |
| `buildDLTStats()` | 大乐透：前后区分栏 |
| `buildSSQStats()` | 双色球：红蓝分栏 |

### 4.4 扩展统计

| 函数 | 说明 |
|------|------|
| `loadDistCharts()` | 分布分析：热力图 + 冷热号 |
| `loadTrendCharts()` | 走势分析：和值跨度 + 奇偶比/大小比 |

### 4.5 高级统计

| 函数 | 说明 |
|------|------|
| `loadAdvScatter()` | 散点图（每位置独立渲染） |
| `loadAdvLLN()` | LLN 运行均值（含 ±1σ/±2σ 置信带） |

### 4.6 图表渲染

| 函数 | 说明 |
|------|------|
| `initChart(dom)` | 根据 `currentTheme` 选择 ECharts 主题 |
| `renderChart(domId, opt)` | 单系列柱状/折线图 |
| `renderGroupedBar(domId, opt)` | 多系列分组柱状图 |
| `renderStackedBar(domId, opt)` | 横向堆叠柱状图 |
| `buildAxisNameOptions(xName, yName, opts)` | 轴名称+位置（Y左上/X右下） |
| `disposeCharts()` | 销毁当前 Tab 所有实例 |

### 4.7 标签映射

| 函数 | 说明 |
|------|------|
| `buildNumLabels()` | 彩种 → 位名映射（表格表头、`buildPosLabels` 复用） |
| `buildPosLabels(fields)` | 字段数组 → 标签数组（图例用） |
| `buildHeatmapLabels(positions, lotType)` | 热力图 y 轴标签 |
| `buildPosName(index, lotType)` | 单位置名称（散点图/LLN 标题） |

## 5. 双主题方案

```css
:root, body[data-theme="dark"] {
    --bg-primary: #0d1117;
    --text-primary: #e6edf3;
    /* ... 全部暗色变量 */
}
body[data-theme="light"] {
    --bg-primary: #ffffff;
    --text-primary: #1f2328;
    /* ... 覆盖为浅色 */
}
```

```js
function themeColor(dark, light) {
    return currentTheme === "light" ? light : dark;
}
// 所有硬编码颜色替换为 themeColor(darkVal, lightVal)
```

切换流程：修改 `body[data-theme]` → 重新注册 ECharts 主题 → dispose 全部实例 → 重新渲染。

## 6. 响应式

- PC：左侧固定导航 + 右侧内容区
- 移动端（≤768px）：顶部导航收缩为汉堡菜单，底部固定 Tab 栏
- 图表容器 `min-height` 保证可交互
