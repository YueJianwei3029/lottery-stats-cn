# 接口设计

> **版本**：v2.0 | **日期**：2026-06-07

---

## 1. 规范

- **前缀**：`/api/{table}`，table ∈ {lottery_dlt, lottery_7xc, lottery_ssq, lottery_pl5, lottery_pl3}
- **格式**：`{"code": 200, "message": "success", "data": ...}`
- **生成方式**：`RouterFactory.build(LotteryConfig)` 自动生成 10 个端点/彩种
- **CORS**：`allow_origins=["*"]`

## 2. 端点一览

### 2.1 数据查询

```
GET /api/{table}/query?date=20260101&end_date=20260131&page=1&page_size=10
```

**参数**：date(默认 20260101), end_date(可选), page(≥1), page_size(1-100，默认 20)

**响应**：
```json
{
  "code": 200,
  "data": {
    "records": [{"draw_num": "2025001", "draw_date": "2025-01-01", "num_1": 5, ...}],
    "total": 3000,
    "page": 1,
    "page_size": 10,
    "total_pages": 300
  }
}
```

### 2.2 基础统计

```
GET /api/{table}/stats?date=20260101&end_date=20260131
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "fields": ["num_1","num_2","num_3","num_4","num_5","num_6","back_1"],
    "freq": {
      "num_1": {"0": 15, "1": 20, ...},
      "num_2": {...}, ...
    },
    "odd_even": {
      "num_1": {"odd": 50, "even": 50}, ...
    },
    "big_small": {
      "num_1": {"big": 48, "small": 52}, ...
    }
  }
}
```

### 2.3 扩展统计（4 标准端点）

| 端点 | 说明 |
|------|------|
| `GET /stats/position` | 热力图数据：`{pos_1: {"0": N, "1": N, ...}, pos_2: ...}` |
| `GET /stats/hot_cold` | 冷热号 Top5：`{hot:[{number,count}×5], cold:[{number,count}×5]}` |
| `GET /stats/period_list` | 和值跨度走势：`{records:[{draw_num,sum_val,span,...}]}` |
| `GET /stats/ratio` | 奇偶比/大小比：`{records:[{draw_num,odd_even_ratio,big_small_ratio,...}]}` |

**参数**：`date`(可选), `end_date`(可选)，不传则取全量

**七星彩特殊**：`hot_cold` 返回 `{"front":{"hot":[...],"cold":[...]}, "back":{"hot":[...],"cold":[...]}}`，前后区分开。

**PL3/PL5**：使用 `pos_stats` 模块，传入 `table` + `fields` 参数。

### 2.4 高级统计

| 端点 | 说明 |
|------|------|
| `GET /stats/advanced/scatter?limit=500` | 散点图：`{positions:{"第1位":[{index,value},...]}}` → 前端标题映射 |
| `GET /stats/advanced/lln?limit=500` | 大数定律：`{positions:{"第1位":[{index,running_avg,lower_1sigma,...},...]}}` |
| `GET /stats/advanced/std` | 标准差+均值：`{stats:[{pos,std_dev,mean},...]}` |
| `GET /stats/advanced/percentile` | 分位数 P25/P50/P75 |

**参数**：`date`/`end_date`(可选), `limit`(scatter/lln 专属，100-2000，默认 500)

**注**：scatter/lln 后端返回 `"第X位"`，前端通过 `buildPosName(idx, currentType)` 映射为各彩种具体位名。

### 2.5 公共

| 端点 | 说明 |
|------|------|
| `GET /` | 重定向 `/frontend/index.html` |
| `GET /api/health` | `{"code":200,"data":{"status":"running"}}` |
| `GET /docs` | Swagger UI |

## 3. 路由生成机制

```python
LOTTERY_CONFIGS = [
    LotteryConfig(key="dlt", table="lottery_dlt", tag="超级大乐透",
                  num_fields=[...], ext_module="app.services.dlt_stats"),
    LotteryConfig(key="ssq", ...),
    LotteryConfig(key="7xc", table="lottery_7xc", tag="七星彩",
                  num_fields=["num_1","num_2","num_3","num_4","num_5","num_6","back_1"],
                  ext_module="app.services.qxc_stats"),
    LotteryConfig(key="pl5", ..., uses_pos_stats=True),
    LotteryConfig(key="pl3", ..., uses_pos_stats=True),
]

for cfg in LOTTERY_CONFIGS:
    app.include_router(RouterFactory.build(cfg))
```
