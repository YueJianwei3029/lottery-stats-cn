## 文档体系基础规则

1. **A0**：AI 永无修改权限，即使用户有明确指令也不行。
2. **A1**：技术文档。静态基线(A1S)需用户确认后修改，动态工作(A1D)AI 自由操作。
3. **A2**：测试。AI 只可添加，不可删除。
4. **A3**：临时/草稿。AI 完全读写。
5. **TMP**：模板，仅首次初始化时使用。

1. 不确定的先问用户，不要擅作主张
2. 涉及删表/清数据/改配置，必须先输出方案经确认
3. 禁止全局批量文本替换（编码安全）
4. 所有文件 UTF-8 无 BOM
5. 每轮对话结束时追加用户原始输入到 `other/ALL_PROMPTS.txt`
