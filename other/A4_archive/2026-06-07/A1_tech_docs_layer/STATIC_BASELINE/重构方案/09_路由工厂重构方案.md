# 路由工厂重构方案

> **定位**：将 5 个高度重复的路由文件改为配置驱动的 RouterFactory，实现"加一项功能，5 个彩种自动拥有"。
> **目标**：删除 5 个独立路由文件，合并为工厂类。

---

## 1. 现状问题

当前 5 个路由文件 `lottery_pl3.py / lottery_ssq.py / lottery_pl5.py / lottery_7xc.py / lottery_dlt.py`：

- 每个文件 50~60 行，90% 代码结构相同
- 加一个新功能（如 randomness）需要在 5 个文件中各加 ~10 行
- 注册端点时容易遗漏彩种或拼错路径

## 2. 目标架构

```
main.py
  └→ LotteryConfig × 5（描述差异：表名/字段/标签/功能列表）
       └→ RouterFactory.build(config)
            └→ 自动生成 APIRouter（含所有端点）
                 ├── /query          ← 配置有 "query" 就生成
                 ├── /stats          ← 配置有 "stats" 就生成
                 ├── /stats/position ← 配置有 "position" 就生成
                 ├── ...             ← 配置有对应 feature 就生成
                 └── /randomness     ← 新增 feature，加一行配置即可
```

## 3. 核心设计

### 3.1 LotteryConfig 数据类

```python
@dataclass
class LotteryConfig:
    key: str              # "dlt" | "ssq" | "7xc" | "pl5" | "pl3"
    table: str            # "lottery_dlt" | ...
    tag: str              # "超级大乐透" | ...
    
    # 统计服务注入（惰性加载）
    stats_module: str     # "app.services.dlt_stats" | ...
    
    # 基础统计参数
    num_fields: list[str]     # ["front_1", ..., "back_2"]
    pos_labels: list[str]     # ["前区1", ..., "后区2"]
    
    # 特殊字段分组（扩展统计用）
    extra_config: dict       # {
                             #   "FRONT_FIELDS": [...], 
                             #   "BACK_FIELDS": [...],
                             #   "BIG_THRESHOLD": 18,
                             #   ...
                             # }
    
    # 功能开关（加一个新功能只需加一行）
    features: set[str]       # {"query", "stats", "position", "hot_cold",
                             #  "period_list", "ratio", "advanced_std",
                             #  "advanced_percentile", "advanced_normal",
                             #  "randomness"}
```

### 3.2 工厂方法签名

```python
# app/core/router_factory.py

class RouterFactory:
    """根据 LotteryConfig 自动生成完整的 APIRouter"""
    
    @staticmethod
    def build(config: LotteryConfig) -> APIRouter:
        """返回一个含所有配置端点的 APIRouter"""
```

内部逻辑：遍历 `config.features`，对每个 feature 用闭包生成路由函数。

### 3.3 功能映射表

工厂内部维护 feature → 处理函数的映射：

| feature | 端点路径 | 调用服务 | 参数 |
|---------|---------|---------|------|
| query | `/query` | db.query(table, ...) | date, end_date, page, page_size |
| stats | `/stats` | base_stats(table, ...) | date, end_date |
| position | `/stats/position` | ext.position_stats(...) | date, end_date |
| hot_cold | `/stats/hot_cold` | ext.hot_cold_stats(...) | date, end_date |
| period_list | `/stats/period_list` | ext.period_list_stats(...) | date, end_date |
| ratio | `/stats/ratio` | ext.ratio_stats(...) | date, end_date |
| advanced_std | `/stats/advanced/std` | std_dev_stats(table, ...) | date, end_date |
| advanced_percentile | `/stats/advanced/percentile` | percentile_stats(table, ...) | date, end_date |
| advanced_normal | `/stats/advanced/normal` | normal_dist_stats(table, ...) | date, end_date |
| randomness | `/randomness` | randomness_test(config, ...) | periods, alpha, date, end_date |

### 3.4 扩展统计模块的惰性加载

```python
def _get_ext_module(config):
    """按彩种动态导入对应的扩展统计模块，避免循环依赖"""
    mapping = {
        "dlt": "app.services.dlt_stats",
        "ssq": "app.services.ssq_stats",
        "7xc": "app.services.qxc_stats",
        "pl5": "app.services.pos_stats",
        "pl3": "app.services.pos_stats",
    }
    return importlib.import_module(mapping[config.key])
```

### 3.5 main.py 变化

```python
# 注册所有路由（5行配置替代5个独立导入）

configs = [
    LotteryConfig(key="dlt", table="lottery_dlt", tag="超级大乐透", ...),
    LotteryConfig(key="7xc", table="lottery_7xc", tag="七星彩", ...),
    LotteryConfig(key="ssq", table="lottery_ssq", tag="双色球", ...),
    LotteryConfig(key="pl5", table="lottery_pl5", tag="排列5", ...),
    LotteryConfig(key="pl3", table="lottery_pl3", tag="排列3", ...),
]

for config in configs:
    app.include_router(RouterFactory.build(config))
```

## 4. 新增功能流程

加了 RouterFactory 后，未来加新功能只需 **3 步**：

```
1. 写 services/new_feature.py（业务逻辑）
2. 在 RouterFactory 里加一个 feature handler（~15行模板）
3. 在 configs 的 features 集合里加一行 "new_feature"
   → 5 个彩种自动拥有新端点
```

对比现状：
```
1. 写 services/new_feature.py
2. 改 lottery_pl3.py 加 ~10行
3. 改 lottery_ssq.py 加 ~10行
4. 改 lottery_pl5.py 加 ~10行
5. 改 lottery_7xc.py 加 ~10行
6. 改 lottery_dlt.py 加 ~10行
```
