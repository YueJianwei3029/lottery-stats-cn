## 1. 技术选型

## 3. 组件设计

| 组件/概念 | 功能 | 说明 |
|-----------|------|------|
| TabBar | 4 个主 Tab 切换，高亮当前选中 | 切换时销毁旧图表、重新调用接口渲染 |
| DatePicker | 起始日期 + 结束日期 + 查询按钮 | 默认 date=20260101，end_date=空 |
| DataTable | 分页表格，展示开奖数据 | loading/data/empty/error 四种状态 |
| Pagination | 分页控件 | 当前页/总页数，触发翻页请求 |
| ChartPanel | 通用图表容器 | 右上角 `?` 按钮弹出统计原理说明 |
| config.js | API 基地址配置 | localhost → `http://localhost:8000/api`，其他 → `/api` |
| dataZoom | ECharts 滑块组件 | 走势图支持 20~200 期范围拖动 |

## 4. 图表清单

### 基础统计 Tab（5 个彩种通用）

| 图表 | 类型 | 数据来源 | 说明 |
|------|------|---------|------|
| 号码频率分布 | 柱状图 | `/stats` → freq | 按位置拆分渲染 |
| 奇偶比例 | 饼图 | `/stats` → odd_even | 奇数/偶数占比 |
| 大小比例 | 饼图 | `/stats` → big_small | 大号/小号占比 |

### 扩展统计 Tab（5 个彩种通用）

**子菜单：[分布分析]**

| 图表 | 类型 | 数据来源 | 说明 |
|------|------|---------|------|
| 号码热力图 | 热力图 | `/stats/position` | 位置×号码频率矩阵 |
| 冷热号 Top N | 横向柱状图 | `/stats/hot_cold` | 热号/冷号对比 |

**子菜单：[走势分析]**

| 图表 | 类型 | 数据来源 | 说明 |
|------|------|---------|------|
| 和值走势 | 折线图 | `/stats/period_list` | 每期和值趋势 |
| 跨度走势 | 折线图 | `/stats/period_list` | 每期跨度趋势 |
| 奇偶比走势 | 折线图 | `/stats/ratio` | 每期奇偶比趋势 |

> 走势图带 dataZoom 滑块，默认显示 20~200 期可调。

## 5. 页面交互流程

```
用户打开 index.html
  → 默认选中"数据查询" Tab
  → 默认彩种 = 排列3
  → 自动调用 /api/lottery_pl3/query
  → 渲染数据表格

用户切换彩种（下拉或按钮）
  → 销毁当前所有图表实例
  → 重新调用对应彩种的 query + stats 接口
  → 重新渲染当前 Tab 内容

用户切换主 Tab
  → 销毁旧图表
  → 调用对应接口渲染
  → 扩展统计 Tab 默认显示"分布分析"子菜单

用户修改日期查询
  → 重新调用 query 接口
  → 刷新表格（统计传入日期参数）

用户翻页
  → 调用 query 接口（page 参数变化）
  → 刷新表格

用户拖动 dataZoom 滑块
  → 调整走势图显示期数范围
  → 本地过滤数据，无需重新请求接口
```

## 6. 文件结构

```
frontend/
├── index.html      # 单页入口，包含 4 Tab HTML 结构
├── css/
│   └── style.css   # 布局、表格、Tab、分页、图表容器、响应式样式
└── js/
    ├── config.js   # 最先加载，设置 window.APP_CONFIG.API_BASE
    ├── app.js      # 主逻辑：Tab切换、接口调用、分页、ECharts渲染、dataZoom
    └── echarts.min.js  # ECharts 5.5
```

## 7. 关键 JS 模块职责（app.js）

| 模块 | 职责 |
|------|------|
| LotteryApp 类 | 前端主控制器，管理 currentType/currentTab/chartInstances |
| switchTab() | 切换 4 个主 Tab，调用对应渲染方法 |
| queryData() | 调用 /query 接口，更新表格 |
| loadCommonStats() | 调用 /stats，渲染基础统计 3 图表 |
| loadExtendCharts() | 调用 4 个扩展端点，渲染热力图+冷热号+走势图 |
| loadDltExtend/loadQxcExtend/loadSsqExtend/loadPosExtend() | 各彩种扩展渲染 |
| renderChart() | 通用 ECharts 渲染 + dataZoom + help tooltip |
| destroyAllCharts() | 切换 Tab 前清理旧实例 |
