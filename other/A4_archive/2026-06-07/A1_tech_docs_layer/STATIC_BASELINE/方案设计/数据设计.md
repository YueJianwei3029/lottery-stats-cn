# 数据设计

> **定位**：数据模型、表结构、关系定义。

---

## 1. 实体关系概览

```
（5 张独立表，无外键关系，各彩种独立存储）

┌───────────┐  ┌───────────┐  ┌───────────┐  ┌───────────┐  ┌───────────┐
│lottery_pl3│  │lottery_ssq│  │lottery_pl5│  │lottery_7xc│  │lottery_dlt│
│  排列3     │  │  双色球    │  │  排列5     │  │  七星彩    │  │ 超级大乐透 │
│ ~7,620行   │  │ ~3,460行   │  │ ~7,620行   │  │ ~3,043行   │  │ ~2,880行   │
└───────────┘  └───────────┘  └───────────┘  └───────────┘  └───────────┘
```

## 2. 表结构

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| draw_num | VARCHAR(10) | UNIQUE, NOT NULL | 期号（如 2026001） |
| draw_date | DATE | NOT NULL | 开奖日期 |

## 3. 索引设计

| 表名 | 索引字段 | 类型 | 用途 |
|------|---------|------|------|
| 全部5表 | draw_num | UNIQUE | 按期号去重，保证数据唯一 |
| 全部5表 | draw_date | 普通 INDEX | 按日期范围查询加速 |

## 4. 统计字段分组（services 层使用）

| 彩种 | 字段组 | 说明 |
|------|--------|------|
| lottery_pl3 | PL3_FIELDS = [num_1, num_2, num_3] | 3位0-9数字 |
| lottery_pl5 | PL5_FIELDS = [num_1..num_5] | 5位0-9数字 |
| lottery_ssq | RED_FIELDS = [red_1..red_6], BLUE_FIELD = [blue_1] | 红球1-33 + 蓝球1-16 |
| lottery_7xc | FIELDS = [num_1..num_7] + [special_num] | 7基本号 + 特别号 1-30 |
| lottery_dlt | FRONT_FIELDS = [front_1..front_5], BACK_FIELDS = [back_1, back_2] | 前区1-35 + 后区1-12 |

## 5. 初始数据

无需初始数据，首次运行爬虫自动填充。
