# 接口设计

> **定位**：API 端点、请求/响应格式、状态码。

---

## 1. 接口列表总览

### 通用端点（5 个彩种均有）

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/{彩种}/query` | 分页查询，支持 date + end_date 区间 |
| GET | `/api/{彩种}/stats` | 基础统计（频率/奇偶/大小） |

### 扩展统计端点（5 个彩种均有）

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/{彩种}/stats/position` | 各位置号码出现频率 |
| GET | `/api/{彩种}/stats/hot_cold` | 冷热号（无日期默认50期） |
| GET | `/api/{彩种}/stats/period_list` | 逐期和值/跨度/奇偶/大小列表 |
| GET | `/api/{彩种}/stats/ratio` | 奇偶比/大小比/012路分布 |

### 高级统计端点（5 个彩种均有）

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/{彩种}/stats/advanced/std` | 各位置标准差 + 均值 |
| GET | `/api/{彩种}/stats/advanced/percentile` | P25/P50/P75 分位数 |
| GET | `/api/{彩种}/stats/advanced/normal` | 正态分布拟合（偏度/峰度/直方图），仅 DLT/SSQ |

### 系统端点

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/health` | 健康检查 → `{"code": 200, "message": "ok", "data": {"status": "running"}}` |
| GET | `/` | 重定向 → `/frontend/index.html` |
| GET | `/docs` | Swagger UI 自动文档 |

### 彩种路径前缀

| 彩种 | 前缀 | 表名 |
|------|------|------|
| 排列3 | `/api/lottery_pl3` | lottery_pl3 |
| 双色球 | `/api/lottery_ssq` | lottery_ssq |
| 排列5 | `/api/lottery_pl5` | lottery_pl5 |
| 七星彩 | `/api/lottery_7xc` | lottery_7xc |
| 大乐透 | `/api/lottery_dlt` | lottery_dlt |

> 端点统一：position/hot_cold/period_list/ratio（4项），高级统计 std/percentile/normal（3项）。
> 总计：5彩种 × (2通用 + 4扩展 + 3高级) + 2系统 = **47 个端点**。

## 2. 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误（如日期格式非法） |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 3. 接口详情

### GET /api/{彩种}/query —— 分页查询

**请求参数（Query）**

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| date | string | 否 | 20260101 | 起始日期 YYYYMMDD |
| end_date | string | 否 | null | 结束日期 YYYYMMDD（不传=至今） |
| page | int | 否 | 1 | 页码（≥1） |
| page_size | int | 否 | 20 | 每页条数（1-100） |

**响应体 data**

```json
{
  "total": 100,
  "page": 1,
  "page_size": 20,
  "records": [
    {
      "draw_num": "2026001",
      "draw_date": "2026-01-01",
      "num_1": 3, "num_2": 7, "num_3": 2
    }
  ]
}
```
（records 字段按各表结构返回，最新数据置顶）

### GET /api/{彩种}/stats —— 基础统计

**请求参数**：date/end_date（可选，不传=全量历史数据）

**响应体 data**

```json
{
  "freq": {
    "num_1": {"0": 45, "1": 52, ...},
    "num_2": {"0": 40, "1": 55, ...},
    ...
  },
  "odd_even": {
    "odd_count": 120, "even_count": 118,
    "odd_ratio": "50.4%", "even_ratio": "49.6%"
  },
  "big_small": {
    "big_count": 130, "small_count": 108,
    "big_ratio": "54.6%", "small_ratio": "45.4%"
  }
}
```

> freq 按位置拆分，奇偶/大小严格遵循各彩种官方分界规则。

### GET /api/{彩种}/stats/position —— 位置频率

```json
{
  "pos_1": {"1": 15, "2": 18, ...},
  "pos_2": {...},
  ...
}
```

### GET /api/{彩种}/stats/hot_cold —— 冷热号

```json
{
  "hot": [{"number": 15, "count": 4}, ...],
  "cold": [{"number": 22, "count": 0}, ...]
}
```
> 无日期参数默认最近 50 期，有日期按范围全量统计。

### GET /api/{彩种}/stats/period_list —— 逐期指标

```json
{
  "records": [
    {"draw_num": "2026001", "sum": 45, "span": 8, "odd": 3, "even": 4, "big": 2, "small": 5},
    ...
  ]
}
```
> 无日期参数默认最近 500 期。

### GET /api/{彩种}/stats/ratio —— 比例分布

```json
{
  "records": [
    {
      "draw_num": "2026001",
      "odd_even_ratio": "4:3",
      "big_small_ratio": "3:4",
      "road_012": {"0": 2, "1": 3, "2": 2}
    },
    ...
  ]
}
```

### GET /api/{彩种}/stats/advanced/std —— 标准差

**请求参数**：date/end_date（可选）

```json
{
  "fields": ["num_1", "num_2", ...],
  "data": [
    {"field": "num_1", "mean": 4.52, "std": 2.87, "min": 0, "max": 9},
    ...
  ]
}
```

### GET /api/{彩种}/stats/advanced/percentile —— 分位数

```json
{
  "fields": ["num_1", "num_2", ...],
  "data": [
    {"field": "num_1", "p25": 2, "p50": 5, "p75": 7},
    ...
  ]
}
```

### GET /api/{彩种}/stats/advanced/normal —— 正态分布

```json
{
  "fields": [...],
  "data": [
    {
      "field": "num_1",
      "mean": 4.52, "std": 2.87,
      "skewness": 0.15,
      "kurtosis": -1.2,
      "histogram": {"bins": [0,1,2,...], "counts": [45,52,...]}
    },
    ...
  ]
}
```
> normal 端点目前仅 lottery_dlt 和 lottery_ssq 实现，其他彩种前端占位。
