# 系统架构

> **定位**：技术选型与项目结构，供开发人员和技术评审使用。

---

## 1. 技术栈

| 组件 | 选型 | 版本 | 用途 |
|------|------|------|------|
| 编程语言 | Python | 3.13 | 爬虫、清洗、接口、调度、统计 |
| Web 框架 | FastAPI | latest | RESTful API 服务 |
| ASGI 服务器 | uvicorn | latest | 运行 FastAPI 应用 |
| 数据库 | MySQL | 8.0 | 结构化存储开奖数据 |
| 数据库驱动 | PyMySQL | latest | Python 连接 MySQL |
| 统计加速 | numpy | ≥ 1.26.0 | 向量化统计计算 |
| 定时调度 | APScheduler | latest | 每 24h 触发爬取任务 |
| HTTP 请求 | requests | latest | 下载外部 TXT 数据 |
| 反向代理 | nginx | alpine | Docker 生产模式统一入口 |
| 图表库 | ECharts | CDN 5.5 | 前端数据可视化 |
| 包管理 | pip + requirements.txt | — | 依赖管理 |

## 2. 架构模式

```
                    浏览器
                      │
                      ▼
              ┌──────────────┐
              │   Nginx :80  │  ← Docker 生产模式
              └──────┬───────┘
         ┌───────────┴───────────┐
         ▼                       ▼
┌─────────────────┐    ┌─────────────────┐
│ /api/* 代理      │    │ /* 静态文件      │
│ → app:8000      │    │ (frontend/)     │
└────────┬────────┘    └─────────────────┘
         │
         ▼
┌─────────────────┐    ┌─────────────────┐
│ FastAPI         │───→│ MySQL :3306     │
│ (5路由+调度器)   │    │ lottery_db      │
└─────────────────┘    └─────────────────┘
```

- **Docker 生产**：3 容器（mysql + app + nginx），`docker compose up -d`
- **本地开发**：FastAPI StaticFiles 模式，`start.bat` 一键启动，无需 nginx
- **API 自适应**：`config.js` 根据 `window.location.hostname` 自动切换本地/云端基地址

## 4. 模块划分

| 模块 | 职责 | 依赖 |
|------|------|------|
| core/config | 集中管理所有配置（DB/数据源/调度/端口） | 无 |
| core/database | 建库建表、连接池、查询/插入封装 | config |
| core/scheduler | 每24h自动执行采集流水线 | database, crawler, cleaner |
| crawler | 下载各彩种公开 TXT 数据，返回原始文本 | config |
| cleaner | 解析 TXT → 结构化 dict，校验有效性 | 无 |
| routers | 5个独立子路由，接收HTTP请求→调用services→返回JSON | database, services |
| services | 执行统计计算（numpy向量化），从数据库读取并计算 | database |
| frontend | 纯HTML页面，调用API，ECharts渲染图表 | routers（HTTP调用） |
| nginx | Docker生产模式反向代理+静态文件 | 无（Docker编排） |

## 5. 部署方式

### 端口分配表

| 服务 | 端口 | 对外 | 说明 |
|------|------|------|------|
| Nginx | 80 | ✅ | Docker生产统一入口 |
| FastAPI | 8000 | ❌ | Docker内部（nginx代理）/ 本地开发对外 |
| MySQL | 3306 | ❌ | 内部通信 |

### 启动命令

**本地开发（Windows）：**
```powershell
# 双击 start.bat 或手动：
pip install -r requirements.txt
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
# 访问 http://localhost:8000/frontend/index.html
```

**Docker 部署：**
```bash
git clone https://github.com/YueJianwei3029/lottery-stats-cn.git
cd lottery-stats-cn
echo "DB_PASSWORD=lottery123" > .env
docker compose up -d
# 访问 http://localhost（本地）或 http://www.yuejw.top（云端）
```
