# 修改方案：正态分布图重新设计

> 日期：2026-06-07
> 决策：只对 DLT 前区(35个号码) 和 SSQ 红球(33个号码) 做正态分布拟合，其他区号码种类太少不做。

---

## 1. 方案概述

### 核心变化

| | 当前实现 | 新方案 |
|------|---------|--------|
| 统计方式 | 按位置逐字段统计频率 | 按号码值**跨位置聚合**统计总频次 |
| 理论依据 | 次序统计量期望 | 中心极限定理：聚合后每个号码出现次数~二项分布→近似正态 |
| 覆盖范围 | 5彩种全部字段 | 仅 DLT 前区 + SSQ 红球 |
| 图表数量 | DLT 7张 / SSQ 7张 | DLT 1张(前区) / SSQ 1张(红球) |

### 统计学原理

- 每个号码在 N 期中的出现次数 = Σ(每期是否出现)，服从二项分布 Binomial(n, p)
- 当 n 足够大时，二项分布近似正态分布 N(np, np(1-p))
- 通过比较实际频次分布 vs 理论正态曲线，可直观判断号码出现是否均匀

## 2. 后端修改

### 2.1 `app/services/advanced_stats.py` — 重写 `normal_dist_stats()`

**当前逻辑**：逐字段统计，每个字段独立返回 mean/std/skewness/kurtosis/histogram

**新逻辑**：对于 DLT/SSQ，跨位置聚合；其他彩种返回空

**新的返回格式**：

```json
{
  "type": "dlt_front",        // "dlt_front" | "ssq_red" | null
  "num_range": [1, 35],       // 号码范围
  "total_periods": 2880,      // 总期数
  "total_draws": 14400,       // 总抽取次数 (期数 × 每期位数)
  "expected_freq": 411.43,    // 理论期望频次 = total_draws / num_count
  "expected_std": 20.07,      // 理论标准差 = sqrt(total_draws × p × (1-p))
  "skewness": 0.15,           // 实际频次分布的偏度
  "kurtosis": -0.32,          // 实际频次分布的峰度
  "histogram": {
    "numbers": [1,2,...,35],  // 号码值 (X轴)
    "counts": [410,398,...]   // 各号码出现总频次 (Y轴)
  },
  "theoretical_curve": {      // 理论正态曲线数据 (供前端画对比曲线)
    "x": [380,382,...],       // X轴采样点
    "y": [0.01,0.015,...]     // 对应概率密度 × 缩放系数
  }
}
```

**关键算法（伪代码）**：

```python
def normal_dist_stats(table, date=None, end_date=None):
    if table == "lottery_dlt":
        zone_fields = ["front_1","front_2","front_3","front_4","front_5"]
        num_count = 35
        result_type = "dlt_front"
    elif table == "lottery_ssq":
        zone_fields = ["red_1","red_2","red_3","red_4","red_5","red_6"]
        num_count = 33
        result_type = "ssq_red"
    else:
        return {"type": None}  # 其他彩种不支持

    records = db.fetch_all(table, date=date, end_date=end_date)
    if not records:
        return {"type": result_type, "num_range": [1, num_count], ...}

    # 1. 跨位置聚合：统计每个号码(1~num_count)的出现总次数
    arr = records_to_array(records, zone_fields)
    all_vals = arr[~np.isnan(arr)].astype(int).flatten()
    counts = np.bincount(all_vals, minlength=num_count + 1)[1:]  # 索引0不用

    # 2. 计算实际统计量
    total_draws = len(all_vals)
    actual_mean = np.mean(counts)
    actual_std = np.std(counts, ddof=1)
    z = (counts - actual_mean) / actual_std
    skewness = np.mean(z ** 3) if actual_std > 0 else 0
    kurtosis = np.mean(z ** 4) - 3 if actual_std > 0 else 0

    # 3. 计算理论值
    p = 1.0 / num_count
    expected_freq = total_draws * p
    expected_std = np.sqrt(total_draws * p * (1 - p))

    # 4. 生成理论正态曲线（x: 频次范围, y: 概率密度×缩放）
    x = np.linspace(expected_freq - 4*expected_std, expected_freq + 4*expected_std, 100)
    y = num_count * np.exp(-0.5 * ((x - expected_freq) / expected_std) ** 2) / (expected_std * np.sqrt(2*np.pi))

    return {
        "type": result_type,
        "num_range": [1, num_count],
        "total_periods": len(records),
        "total_draws": total_draws,
        "expected_freq": round(expected_freq, 2),
        "expected_std": round(expected_std, 2),
        "skewness": round(float(skewness), 3),
        "kurtosis": round(float(kurtosis), 3),
        "histogram": {
            "numbers": list(range(1, num_count + 1)),
            "counts": [int(c) for c in counts],
        },
        "theoretical_curve": {
            "x": [round(v, 2) for v in x],
            "y": [round(v, 4) for v in y],
        },
    }
```

## 3. 前端修改

### 3.1 `frontend/js/app.js` — 重写 `loadAdvNormal()`

**当前逻辑**：逐字段渲染多张图，每张图 = 柱状图(频次) + 红色拟合曲线

**新逻辑**：仅 DLT 和 SSQ 各渲染 1 张图，包含：
- 柱状图：各号码的实际出现频次
- 理论正态曲线（虚线）：基于理论期望和理论标准差
- 实际正态拟合曲线（实线）：基于实际数据的 mean/std
- 图例：显示偏度/峰度/理论期望等统计量

**前端渲染逻辑（伪代码）**：

```javascript
function loadAdvNormal(container, dateParams) {
    // ... fetch 数据 ...

    if (!data.type) {
        container.innerHTML = '<占位：当前彩种不支持正态分布分析>';
        return;
    }

    // 单张图表
    var chart = initChart(wrap);
    chart.setOption({
        title: { text: data.type === 'dlt_front' ? '大乐透前区正态分布' : '双色球红球正态分布' },
        tooltip: { trigger: 'axis' },
        legend: { data: ['实际频次', '理论正态曲线', '实际拟合曲线'] },
        xAxis: {
            type: 'category',
            data: data.histogram.numbers.map(String),
            name: '号码'
        },
        yAxis: { type: 'value', name: '出现频次' },
        series: [
            // 柱状图：实际频次
            { name: '实际频次', type: 'bar', data: data.histogram.counts,
              itemStyle: { color: 'rgba(88,166,255,0.55)' } },
            // 理论正态曲线（虚线，基于理论期望/标准差）
            { name: '理论正态曲线', type: 'line', smooth: true,
              data: data.theoretical_curve.y.map(...), // 映射到号码X轴
              lineStyle: { color: '#3fb950', type: 'dashed', width: 2 } },
            // 实际拟合曲线（实线，基于实际 mean/std）
            { name: '实际拟合曲线', type: 'line', smooth: true,
              data: fittedCurve,
              lineStyle: { color: '#f85149', width: 2 } },
        ],
        graphic: [
            // 底部统计量文本
            { type: 'text', ...,
              style: { text: 'μ(理论)=' + data.expected_freq +
                     ' σ(理论)=' + data.expected_std +
                     ' 偏度=' + data.skewness +
                     ' 峰度=' + data.kurtosis } }
        ]
    });
}
```
