# 2026-06-07

## 第二轮修复：横坐标截断 + 热力图样式 + 移动端滑动

### 问题
1. 所有图表 x 轴标签被截断（grid.bottom 不够大）
2. 热力图在深色主题下太亮，数字重叠显示
3. 热力图纵坐标用"位置1""位置2"不合适
4. 移动端图表不能横向滑动

### 修复

#### 1. 横坐标截断 (app.js 多处)
- `renderChart`: bottom 40→55/70（有旋转时更大）
- `renderGroupedBar`: bottom 50→60/80（有旋转时更大）
- `renderStackedBar`: bottom 50→60
- 热力图 grid bottom: 75→80，left 70→80
- 走势图 grid bottom: 70→85
- 高级统计 scatter/LLN/normal: bottom 40→50/55
- 所有 x 轴旋转标签加 fontSize:10

#### 2. 热力图修复 (app.js)
- visualMap 色阶从默认亮色改为深色友好色阶：`["#161b22","#1a3a5c","#1e5a8e","#2980b9","#3498db","#5dade2","#85c1e9"]`
- label.show 从 true 改为 false（默认不显示数字）
- emphasis.label.show = true（hover 时显示）
- tooltip 格式化显示完整位置名+号码

#### 3. 热力图纵坐标 (app.js)
- 新增 `buildHeatmapLabels()` 函数，按彩种映射：
  - 大乐透: 前区1~5, 后区1~2
  - 双色球: 红球1~6, 蓝球
  - 排列三: 百位/十位/个位
  - 排列五: 万位/千位/百位/十位/个位
  - 七星彩: 第1位~第7位
- `buildPosLabels` 中 num_1~7 标签从"位置1"改为"号码1"

#### 4. 移动端横向滑动 (style.css)
- `#statsContainer`, `#extendCharts`, `#advancedCharts` 容器设置 overflow-x:auto
- 单列图表 min-width: 480~500px
- 并排图表（stats-bottom, chart-half-row）改为 display:flex，各 min-width: 340px
- 子菜单 flex-wrap: nowrap + overflow-x: auto
- 保留移动端图表高度 280px
